# 流程模型 — M1a 模型配置管理

## 1. 流程总览

| 流程 ID | 流程名称 | 触发方式 | 涉及行为 |
|---------|---------|---------|---------|
| P-001 | 查看模型配置 | 管理员打开配置页面 | B-001 |
| P-002 | 修改模型配置 | 管理员提交修改表单 | B-002 |
| P-003 | 模型参数构造 | AI 对话请求触发 | B-003, B-004 |

---

## 2. 流程详细定义

### P-001：查看模型配置

**描述：** 管理员打开模型配置页面，查看当前所有模型的配置信息
**触发条件：** 管理员访问配置页面

```
管理员打开配置页面
    │
    ▼
┌─────────────────────────┐
│ [R-008] 管理员权限校验    │  ← 框架层 DRF IsAdminUser
│ 非管理员 → 403           │
│ 前端：非管理员看不到入口   │
│ URL 直接访问 → 错误页面   │
└──────────┬──────────────┘
           │ 通过
           ▼
┌─────────────────────┐
│ B-001 查询所有模型    │
│ GET /api/v1/models/  │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ [R-004] 解密 API Key │  ← 后端内部，为脱敏做准备
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ [R-005] 脱敏 API Key │  ← 返回前处理
│ sk-t****cxyz         │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 前端展示配置卡片      │
│ 按类型分两张卡片:     │
│ - language 配置卡片   │
│ - embedding 配置卡片  │
│ 选填参数 NULL 显示为  │
│ "未设置"              │
└─────────────────────┘
```

---

### P-002：修改模型配置

**描述：** 管理员修改模型参数并保存，配置立即生效
**触发条件：** 管理员提交修改表单

```
管理员编辑模型参数，点击保存
    │
    ▼
┌──────────────────────────┐
│ 前端校验（必填字段非空）    │  ← 前端初步拦截
└──────────┬───────────────┘
           │ 通过
           ▼
┌──────────────────────────┐
│ PUT /api/v1/models/:id/   │
│ 提交修改数据               │
└──────────┬───────────────┘
           │
           ▼
┌──────────────────────────┐
│ [R-008] 管理员权限校验     │  ← 框架层 DRF IsAdminUser
│ 非管理员 → 403            │
└──────────┬───────────────┘
           │ 通过
           ▼
┌──────────────────────────┐
│ B-002 修改模型配置         │
│                           │
│ ① [R-007] type 字段不可改  │
│ ② [R-001] 必填字段校验     │
│   （PUT 时 6 个，不含 type）│
│   api_key 最少 12 字符     │
│   （脱敏值含 **** 视为有效）│
│ ③ [R-002] 选填参数范围校验  │
│   整数字段不接受小数        │
│ ④ [R-003] NULL/0 区分处理  │
│   （FR-006 存储语义）       │
│ ⑤ [R-005] API Key 脱敏检测 │
│    若传入值含 **** → 保留原值│
│ ⑥ [R-004] API Key 加密     │
│ ⑦ 写入数据库               │
│ ⑧ [R-006] 即时生效         │
└──────────┬───────────────┘
           │
       ┌───┴───┐
       │成功    │失败
       ▼       ▼
  返回更新后   返回校验
  的配置数据   错误信息
  (api_key    (字段 +
   已脱敏)    错误原因)
```

---

### P-003：模型参数构造

**描述：** AI 对话请求时，从数据库读取最新模型配置，构造 API 请求参数
**触发条件：** 用户发起 AI 对话

```
用户发送消息，触发 AI 对话
    │
    ▼
┌─────────────────────────┐
│ B-003 获取激活模型配置    │
│ 查询 type='language'     │
│ AND is_active=true       │
└──────────┬──────────────┘
           │
           ▼
┌─────────────────────────┐
│ [R-004] 解密 API Key     │  ← 获取明文密钥用于调用
└──────────┬──────────────┘
           │
           ▼
┌─────────────────────────┐
│ B-004 构造请求参数        │
│                          │
│ [R-003] NULL/0 区分      │
│ （FR-007 调用行为）:      │
│ - temperature=NULL → 不传 │
│ - temperature=0    → 传 0 │
│ - temperature=0.7  → 传   │
└──────────┬──────────────┘
           │
           ▼
┌─────────────────────────┐
│ 调用模型 API              │
│ POST model.url            │
│ Headers: Authorization    │
│ Body: 构造好的参数         │
│ 注意: 使用 max_context_   │
│ window 原始值（M1a）      │
└─────────────────────────┘
```
